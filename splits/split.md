# vepiset_spike_sleep_video 数据集 Train/Val 划分说明

## 概述

- **数据文件**: `vepiset_spike_sleep_video_with_sleep.csv`
- **总样本数**: 25,449
- **总患者（EEG）数**: 84
- **划分方式**: 按患者（EEG号）整体划分，同一患者的所有样本归属同一集合
- **优化算法**: 模拟退火，目标为各 target 类别和 sleep_status 在 train/val 中比例尽可能接近 80%/20%
- `train_val = 0` 表示训练集，`train_val = 1` 表示验证集
